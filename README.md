# Typescript and Next.js

Next.js implemented using Typescript and Jest

# Installation

Clone the repository

```
git clone https://github.com/maxfouquet/next.js-typescript-starter.git
```

Set up dependencies

```
npm install
```

Development mode

```
npm run dev
```

Start

```
npm start
```

Test

```
npm run test
```

Test Watch

```
npm run test:watch
```

Build

```
npm run build
```

Next.js is running on http://localhost:3000
